AI 내부통제 성격유형 테스트

1. index.html을 더블클릭하면 바로 실행됩니다.
2. 인터넷 연결 없이도 사용할 수 있습니다.
3. GitHub Pages, Netlify, 사내 웹서버 등에 index.html을 올리면 링크 형태로 배포할 수 있습니다.
4. 응답은 서버에 저장되지 않으며 사용자의 기기 안에서만 계산됩니다.

유형 계산
- A 선택: 원칙수호형
- B 선택: 공감소통형
- C 선택: 근거분석형
- D 선택: 균형조정형
- 가장 많이 선택한 유형을 결과로 표시
- 동점이면 공동 유형으로 표시
